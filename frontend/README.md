# Notiva Frontend

Notiva is a React + Vite frontend for a smart note-taking web application. It connects to the Django REST backend and provides login, registration, note creation, search, filters, folders, reminders, trash, export, and settings.

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Local URL

```text
import.meta.env.VITE_API_URL
```

The backend should be running on:

```text
import.meta.env.VITE_API_URL
```

## Main Screens

- Login and registration
- Dashboard
- Create and edit note editor
- Search and filters
- Folders
- Trash
- Settings
